General

About

A fork of discord.py. PyCord is a modern, easy to use, feature-rich, and async ready API wrapper for Discord written in Python.

Key Features

  • Modern Pythonic api using async/await syntax
  • Sane rate limit that prevents 429s
  • Command extension to aid with bot creation
  • Easy to use with an object oriented design
  • Optimised for both speed and memory

Installation

Note: This is documenting the installation of the latest stable release, discord.py v1.7.3. To install the alpha version of PyCord, click here.

Fresh Installation

Note: Python 3.8 or higher is required to install PyCord. Download the latest version of Python here.

  1. Install using pip install py-cord in the command line.

Warning: Discord.py is incompatible with PyCord. If you have discord.py installed, migrate here.

Migration

  1. Uninstall discord.py using pip uninstall discord.py -y, and pip uninstall discord -y
  2. Install PyCord using pip install py-cord

Replit Installation

Warning: Replit is not a good IDE. Its online, its free, but you should definitely not use it to host your bot. When using replit, some of your files might get deleted, and making a big bot will make your bot slower.

  1. Create a file named .replit.
  2. In .replit, copy the code below:
    3. language="python3" run = """ pip install py-cord python main.py """ [packager] ignoredPackages=["discord.py", "discord"]

Note: You might need to change some things to fit your preference.

Alpha Version (v2.0)

Warning: The alpha version is unstable, and should not be used in production.

The alpha version includes support for slash commands, buttons, and context menus. Installation of the alpha is shown below.

  1. Install Pycord using pip install -U git+https://github.com/Pycord-Development/pycord in the command line

Warning: To install the alpha, you need to have Git installed for the alpha to be installed.
